Javascript plug-in for testing responsive designs

  • Go to site

Objective

To build a plug-in tool with javascript that could be used on a site we were working on. Since I was working on a responsive site, I chose to build a tool to create a pop-up window the size of a different device so you could see how the layout looked at that size.

Implementation

I used FontAwesome icons to represent the device sizes, and jQuery to create the hide/show effect on the menu.

It took a lot of trial and error to come to get the device withs and sizes working properly, but the code itself is pretty simple: 

openNewWindow: function(){
						//new window features, size, options...etc
						//35px added to width to accomodate scroll bar and outer border of window
						var newPop;
						var currentURL= window.location.href;
						var maxW = screen.width;
						var maxH = screen.height;
						var newWinSpecs = "scrollbars=yes,status=yes,height=1,width=1";

					$('#desktop').click (function(){
						if(window.isPopup===undefined){
						console.log("you just clicked on the desktop!");
						newPop = window.open(currentURL, "", newWinSpecs);
						newPop.resizeTo(maxW, maxH);
						newPop.isPopup = true;
						}
						else{
						window.resizeTo(maxW, maxH);
						}
					});

It was really satisfying when the I got the tool to work properly. Javascript was really challenging to learn bacause I had been so used to how concrete HTML and CSS are as languages. The abstract nature of javascript was hard to adjust to, but I recognize how much power it has. I'm looking forward to learning more.

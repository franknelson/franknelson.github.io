Responsive informational site

  • Go to Site

Objective

Using an existing responsive grid system, build a four-page responsive site dedicated to a public service, with each page having a different layout.

Implementation

I chose Skeleton as my grid system. Skeleton uses up to 16 columns, but I used 12. This allowed me to use 6, 4, and 3-column layouts to differentiate each page.

The header, footer, nav and image/video clear all the columns on each page, creating consistency in the layout while the number of column change. The footer and nav are set to cover the full width of the browser window. This makes the layout feel like it's part of the larger viewing area, even though it's centered horizontally.

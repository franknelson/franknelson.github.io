Javascript random color experiment

  • Go to site

Objective

I wanted to see if I could build a javascript function that would randomly generate an RGBA value that I could then assign to different elements in the DOM.

Implementation

I figured out that I needed to assign each RGB a random number between 0-255 and then concatenate them so it could be pushed into the CSS: 

var app = {

					colorMachine: function() {
						var R = getRandomInt(0, 255);
						var G = getRandomInt(0, 255);
						var B = getRandomInt(0, 255);
						var randomColor = (R + "," + G + "," + B + "," + 1);
						var newColor = "rgba(" + randomColor + ")";
						return newColor;
					},

Then I used jQuery to assign the five background columns the color values, and set them to change at a random interval between 0.5 and 2 seconds: 

timeMachine:function(){
					var T = getRandomInt(5, 20);
					var newT = (T+"00");
					return newT;
				},

			pageStyles: function(){
				$("div #1 h1").html(app.colorMachine());
				$("div #1").css({
				'background-color': app.colorMachine()
				});

Next steps

It works pretty well now, but I want to make it more modular. Right now it needs to hook into div #ID's to work, but I'd like to get it to the point where you could choose the number of columns and the code would automatically create them.
